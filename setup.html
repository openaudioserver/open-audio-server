Open Audio Server

Your server is running but the Audio Station web interface has not loaded so you can currently only use the iPhone and Android apps to access the server

  • Android

    Download the DS Audio app from:

    • Synology Download Center
    • Play Store

  • iOS

    Download the DS Audio app from:

    • iOS App Store

  • Web

    Audio Station's web interface requires files copyrighted to Synology that cannot be distributed with Open Audio Server.

    To copy these files requires either a Synology NAS or an Xpenology virtual machine running Synology DSM. There are premade Xpenology virtual machines for VMWare Workstation (paid) and VMWare Player (free) and guides on how to install in your own virtual machine but you have to find them yourself.

    Once you have DSM running, either way, Audio Station must be installed from the "Package Manager", and SSH must be enabled in the "Control Panel" under "Terminal & SNMP".

    scp -r administrator@1.2.3.4:/usr/syno/synoman/ /some/where

    Then open Audio Station in your web browser by right clicking the icon in DSM and selecting launch in new window.

    Find your browser setting to "View source" and save all the text into `dsaudio.html`.

    Update your server settings

    • $ export SYNOMAN_PATH=/some/where/synoman
    • $ export DSAUDIO_HTML_PATH=/some/where/dsaudio.html
